\begin{itemize}

\begin{frame}{Assumptions}
\begin{itemize}
\item You are not required to take this class.
\uncover<2->{
\begin{itemize}
\item This assumption is also an answer to most complaints.
\item I hope the class has self-selected the elite.
\end{itemize}
}
\item You are dexterous with C.
\uncover<3->{
\begin{itemize}
\item You ought be able to pump out working C pretty quickly...
\item ... and be able to recognize any C construct.
\item A copy of {\it The C Programming Language} is recommended.
\end{itemize}
}
\item You have a 64-bit x86 Linux box easily available.
\uncover<4->{
\begin{itemize}
\item I can't {\it require} you to install Linux, but...
\item Why aren't you running open source?
\item This class will regularly refer to and modify GNU/Linux source code.
\end{itemize}
}
\end{itemize}
